# Python Project Template

A minimal template for starting a new Python project.

## Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Or with virtual environment
python -m venv .venv
source .venv/bin/activate  # On Linux/Mac
pip install -r requirements.txt
```

## Development

```bash
# Run tests
pytest

# Format code
black src/

# Type check
mypy src/
```
